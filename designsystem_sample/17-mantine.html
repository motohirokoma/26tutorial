Brand documented

Mantine

Mantine is a React component library covering 100+ components and 50+ hooks. Built around CSS variables and a `theme` object that mirrors the SCSS-style 0-9 color shade scale used by Open Color, with bright blue as the default primary.

Comprehensive

100+ components, 50+ hooks. Form / Dates / Notifications / Modals as separate packages.

Themeable

CSS variables expose every color shade, radius, font, and shadow at runtime.

Dark-ready

Built-in light / dark color schemes via <MantineProvider> and useColorScheme.

Color documented Official ↗

Mantine adopts the Open Color palette: 10 shades per hue (0-9, where 0 is lightest, 9 is darkest). Default `primaryColor` is blue, default `primaryShade` is 6 in light mode and 8 in dark mode.

Typography documented Official ↗

Default font is Greycliff CF; Inter is documented as the recommended fallback. Title sizes h1-h6 are exposed via theme.headings, body text uses the same family at 14-16px in the default theme.
