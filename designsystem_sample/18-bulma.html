Bulma

Bulma is a Flexbox-based CSS framework written in Sass. Built without JavaScript dependency, it ships modular CSS classes with a recognizable rainbow palette of vivid turquoise, blue, green, yellow, and red — branded color names mapped directly to semantic intent.

CSS-only

Zero JavaScript dependency — the framework provides class hooks; behaviors are user-implemented.

Flexbox

.columns / .column powers responsive layout via modern Flexbox at every breakpoint.

Modular Sass

Import only the modifiers, components, and utilities you need; customize via Sass variables.

Typography documented Official ↗

Bulma uses the system font stack — BlinkMacSystemFont, -apple-system, 'Segoe UI', Roboto. Titles come in 6 sizes (.title.is-1 through .title.is-6); body sizes use .is-size-1 through .is-size-7.

Form documented Official ↗

Bulma forms use .field, .label, .control, and .input. Modifiers .is-rounded, .is-danger, .is-success drive shape and validation state. Subtle inset box-shadow gives every input a soft 3D feel.

Surfaces documented Official ↗

Card composes .card-image, .card-content, .card-footer. Box is a simpler container with soft shadow. Modal overlays support .modal-card with header / body / footer.

Card

Card with gradient
