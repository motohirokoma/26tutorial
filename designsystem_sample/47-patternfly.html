Brand documented

PatternFly

PatternFly is the open-source design system created by Red Hat to power enterprise applications — most prominently the Red Hat OpenShift console. It emphasizes information density, accessibility (WCAG 2.1 AA baseline), and patterns proven in operations-heavy workflows: forms, tables, dashboards, and wizards. Conservative 3-4px corner radii and tight spacing scales reflect Red Hat's enterprise heritage.

Open

Apache 2.0 / MIT license; design tokens, React components, and HTML/CSS distributed publicly.

Enterprise-grade

Patterns for clusters, clouds, and consoles — built around dense data and operational workflows.

Accessible

WCAG 2.1 AA baseline; focus rings, keyboard navigation, and ARIA patterns are first-class.

Token-driven

Three-tier token model: global → semantic → component, enabling theming and white-label.

Form documented Official ↗

PatternFly forms favor a compact 36px input height, 1px borders, and a 1px focus ring (border + box-shadow). Required fields use a red asterisk; validation states are color-coded green/red/blue.
