shadcn/ui

shadcn/ui is not a component library — it's a collection of re-usable components you copy directly into your project. Built on Radix Primitives for accessibility and Tailwind CSS for styling, it gives you full ownership of the source code and complete control over how every component looks and behaves.

Open code

No black box. The CLI scaffolds component source directly into your repo.

Accessible

Radix Primitives handle keyboard, focus, ARIA, and screen reader behaviour out of the box.

Themeable

CSS variables drive every token. Swap themes (Zinc, Slate, Stone, Gray, Neutral) instantly.

Typography documented Official ↗

shadcn/ui pairs Inter (default) or Geist Sans with a tight letter-spacing scale on display sizes. The typography component documents h1-h4, lead, large, small, muted, and blockquote utility classes.

Buttons documented Official ↗
