HeroUI

HeroUI is a modern React component library that succeeded NextUI in January 2025. It targets v3 with a vibrant gradient identity, rounded geometry (12-16px default radius), and an Inter-based type system. Because docs were only homepage-verified, this page is marked inspired.

Modern

Larger radii (12-16px), gradient surfaces, and Inter geometry feel contemporary.

React-first

Built atop React Aria for keyboard, screen-reader, and focus correctness by default.

Themed via Tailwind

Tailwind plugin model — utility classes compose with the component theme.
