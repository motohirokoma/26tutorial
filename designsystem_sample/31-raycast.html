Brand inspired

Raycast

Raycast is a productivity-focused command palette for macOS. Its aesthetic favors pitch-black surfaces, a single fiery red accent, and crisp system typography. Speed-first interactions, keyboard primacy, and high-density data are core principles.

Fast

Keyboard-first navigation, instant fuzzy search, no chrome. Every interaction is one keystroke away.

Focused

Minimal palette: black surfaces, white text, one accent. Visual noise is the enemy of flow.

Extensible

An open extension API encourages tooling that fits the same compact, dark, command-driven mental model.

Color inspired Official

Raycast palettes drive almost everything from a single fiery red against pitch-black surfaces. Accent colors (cyan, green, blue, yellow, purple) appear sparingly to tag categories.
