Brand inspired

Linear

Linear's product surface is the de-facto reference for modern issue trackers. Signature elements: refined dark surfaces, purple accent (#5E6AD2), command-bar fluency, and unusually high information density without feeling crowded.

Speed

Keyboard-first. Every screen is reachable in under 200ms — instant navigation is part of the brand.

Density

Compact rows, tight tracking. Linear's UI assumes power users with high screen literacy.

Calm

Restrained palette. Purple appears only on focus, primary actions, and critical accents.

Crafted

Hairline borders, subtle gradients, micro-shadows. Detail-obsessive without ornament.

Color inspired Reference ↗

Linear's surface stack starts at near-black #08090A and steps up through panel, sub-panel, and elevation tiers. The brand purple #5E6AD2 is reserved for primary actions, focus rings, and the signature gradient on the marketing site.

Surfaces inspired Reference ↗

Linear uses three surface tiers — panel, sub-panel, elevation — separated by hairline borders rather than heavy shadows. Modals and command bar use the only meaningful drop shadow.
