Base Web

Base Web is Uber's open-source React component library — the web expression of the broader Base Design System. It's optimized for data-dense ride-hailing dashboards: a near-monochrome palette, a single confident blue accent, and modest 4–8px corner rounding. Layouts breathe through whitespace rather than ornament.

Confident

Blue 400 (#276EF1) carries primary action; mono carries everything else.

Functional

Components are tuned for data tables, forms, and operational consoles.

Themable

A token-driven theme provider lets each Uber surface inherit the same primitives.

Conservative shape

Most surfaces use 4–8px radii; pills are reserved for tags, avatars, and toggles.

Color documented Official ↗

Base Web's palette is built on three families: a single accent blue, a 10-step mono ramp, and minimal semantic colors for success / warning / negative. The aesthetic is closer to a black-and-white photo with a single splash of brand blue.

Buttons documented Official ↗

Base Web offers Primary, Secondary, Tertiary, and Minimal kinds. Primary is mono-1000 (true black) — accent blue is reserved for links and the brand surface. 2px corner radius keeps the geometry quietly Uber.
