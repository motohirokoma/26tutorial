Brand documented

tailwindcss

Tailwind CSS is a utility-first framework built around composability. Instead of writing CSS, you compose styles directly in markup using thousands of small, focused class names. The ecosystem includes Tailwind UI, Heroicons, Headless UI, and a meticulously curated color palette that has shaped much of modern web design.

Utility-first

Compose UIs from thousands of single-purpose classes — no naming, no context switching.

Constraint-driven

A predefined design system (spacing, color, typography) prevents inconsistency.

Tree-shaken

JIT compiles only the classes you use — production CSS stays remarkably small.

Buttons documented Official ↗

Tailwind UI button patterns are utility compositions, not pre-built components. Defaults: rounded-md (6px), font-semibold, px-4 py-2.5, with focus-visible ring-2 ring-offset-2.
