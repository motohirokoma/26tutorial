Brand documented

Chakra UI

Chakra UI is a modular React component library prioritizing developer experience, accessibility, and theming via style props. Its teal-led palette and consistent 4px / 6px shape rhythm produce friendly, readable interfaces with minimal customization.

Style Props

Set spacing, color, and layout inline with semantic shorthand props.

Accessible

WAI-ARIA practices baked in — focus rings, keyboard, screen reader.

Theme-able

extendTheme() composes tokens, semantic tokens, and component recipes.

Iconography documented Official ↗

Chakra exposes <Icon> primitives that wrap any SVG and apply size, color, and viewBox tokens. Recommended pairings include react-icons, Lucide, or Tabler Icons. Default size 1em (= 16-24px contextual).

Buttons documented Official ↗

Chakra Button has five variants: solid, outline, ghost, link, and unstyled. Default colorScheme is teal (gray when used in dialogs).
