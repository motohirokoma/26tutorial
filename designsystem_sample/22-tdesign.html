Branddocumented

TDesign

TDesign is Tencent's open-source enterprise design system. It centralizes tokens, components, and ecosystem libraries (Vue / React / Mobile / Miniprogram) used across Tencent's B-end products.

Reliable

Predictable behaviour and visual consistency across products.

Efficient

Token-driven design and dense data display optimized for power users.

Easy

Familiar interaction conventions tuned for Chinese-language content.

Flexible

Fine-grained tokens make rebranding straightforward.
