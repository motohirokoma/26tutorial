ようこそ！Claude Code って何?

あなたの隣に「頼れる相棒」がやってくる、その正体を覗いてみましょう。

所要時間: 30min Phase: Basics 難易度: 入門
ようこそ

はじめまして。これから10時間、一緒に楽しんでいきましょう。この章は30分。難しい話は出てきません。焦らず、わからなくなったら何度でも戻ってOK。今日はまず「Claude Code ってそもそも何?」をふんわり掴むだけで十分です。

2026年1月時点の情報

本章の記述は 2026年1月 時点のものです。Claude も Claude Code も進化が早いため、画面や細かな操作は少し違うことがあります。困ったら公式ドキュメントも覗いてみてください。

Claude Code ってどんな存在?

ひとことで言うと、Claude Code は 「あなたの隣に座ってくれる、頼れる相棒AI」 です。

難しく考えなくて大丈夫。こんなイメージを持ってください。

3つのアナロジー(イメージ)
どれを気に入ってもOK。この章では「新入社員」で進めます。

あなたはこの「新入社員」に日本語でお願いするだけ。Claude Code は、文章を書いたり、HTML や CSS を書いたり、間違いを直したりしてくれます。

「プログラミングの知識が要るんでしょ?」と思うかもしれません。ほぼ要りません。このチュートリアルでも、コードを書くのは Claude Code。あなたは 「何を作りたいか」を伝える人 です。

ここで戸惑っても大丈夫。この章を読み終える頃には、「へぇ、そういうものか」となっている はずです。もう一度読んでもいいし、後から戻ってきてもOK。

もっと詳しく: 「AI」ってそもそも何?

AI(人工知能)は、膨大な文章を読み込んで「次にどんな言葉が来そうか」を予測する仕組みです。人のように「理解」しているわけではないのですが、私たちの質問に対して自然で役立つ答えを返してくれます。Claude もその一種で、Anthropic 社が作っています。

細かい仕組みは知らなくて大丈夫。「賢い文章製造機」くらいのイメージで十分です。

他のAIツールと何が違うの?

「チャットAI ならもう使ってるよ」という方もいるでしょう。では、Claude Code は何が違うのか。ざっくり言うと 「あなたのPCで実際に手を動かしてくれる」 ところです。

非エンジニアでもできる、3つの理由

「でも私、プログラミングできないし...」と身構えなくて大丈夫。今のあなたに必要なのは、日本語で丁寧にお願いする力 だけです。

  • 理由1: 日本語でOK — プログラミング言語 を覚える必要はありません。「こういうのを作って」「ここを直して」と、人にお願いするのと同じです。
  • 理由2: 何度でもやり直せる — 失敗しても、元に戻せます(Ch05で習う Git を使えば完璧)。だから怖くない。
  • 理由3: 相棒が優しい — Claude Code は、わからないことを聞くと丁寧に説明してくれます。恥ずかしがらず、何度でも聞いてOK。

この3つがあれば、非エンジニアでも 10時間で自分のWebアプリを作って世界に公開する ところまでいけます。すごくないですか?

「コンピュータに詳しくなる」のではなく、「伝え方が上手な人」になる。それが Claude Code 時代の学び方です。

10時間で作るもの:「お気に入りリスト」

ゴールは、自分の「お気に入りリスト」Webアプリ を作って、世界中からアクセスできるURLを手に入れることです。

MKからのメッセージ

わたし(MK)もエンジニアの専門教育を受けたわけではありません。Claude Code と仲良くなる方法は、「わからなくても先に進む」「わかった気でOK」。これが本当に大事です。

小さな成功: 「Claude Code は○○できる」を3つ言ってみよう

今日の 小さなゴール(ハンズオンの代わり)は、これだけ:

できたら嬉しい

自分の口で3つ言える

  • 「Claude Code は○○できる」
  • 「Claude Code は○○できる」
  • 「Claude Code は○○できる」
ヒント

3つの候補

  • ファイルを読んでくれる
  • コードを書いてくれる
  • エラーを一緒に直してくれる

紙でも、メモアプリでも、LINEの自分宛でもOK。書き出してみることが大事。書けたら、もうあなたは「Claude Code を説明できる人」です。

もっと詳しく: できることのサンプル(ひとまず眺めるだけで可)
  • TODOリストをテキストファイルから読み上げて、整理してくれる
  • 日記の誤字を直してくれる、別の言い回しを提案してくれる
  • HTMLファイルを新しく作ってくれる
  • エラーが出たとき「どこが悪い?」と聞けば一緒に探してくれる
  • 「このフォルダのファイル一覧を教えて」と頼めば答えてくれる

全部わからなくて大丈夫。このチュートリアルで1つずつ体験します。

今日のエクササイズ

Exercise 1
入門

「Claude Code って何?」を誰かに説明してみる

家族でも友達でも、自分の日記でもOK。60秒で説明してみてください。

  1. 相手(または自分)を決める
  2. 「Claude Code は、〇〇 してくれる AI です」と一文で言う
  3. 気に入ったアナロジー(新入社員/相棒/魔法の箱)を1つ添える
模範解答を見る

例:

「Claude Code は、私のパソコンの中に住んでる 頼れる相棒AI です。日本語でお願いすると、ファイルを作ったり、文章を書いたり、間違いを直したりしてくれます。優秀な新入社員が隣に座ってくれるイメージです」

うまく言えなくても大丈夫。声に出して言ってみた こと自体がゴールです。

Exercise 2
応用

いまのあなたは

Claude Code が 「頼れる相棒AI」 で、日本語でお願いするだけで動いてくれる存在だとわかりました。非エンジニアでも10時間で自分のWebアプリが作れる、そのイメージを掴めた状態です。これで Claude Code の世界への第一歩 が完了。焦らず、次は15分で相棒を自分のPCに呼び寄せましょう。今日ここまで読んだあなたは、もう十分「始めた人」です。自信を持っていい。
