はじめての方へ (前提知識)

プログラミング未経験の方が本編に入る前に、最低限おさえておきたい用語と操作をまとめています。流し読みで大丈夫。

所要時間: 30-45min Phase: Prerequisites 難易度: 準備

すべてを暗記する必要はありません。「こんな言葉があったな」と覚えておいて、本編で出てきたときにここに戻れれば十分です。わからない言葉は ツールチップ に説明があります。

この章で押さえる4つの土台
この5つを「なんとなく」知っておくだけで、本編がぐっと読みやすくなります。

ターミナルってなに?

ターミナル は「文字でパソコンに指示を出す画面」です。普段使っているマウスの代わりに、キーボードで命令を打ちます。Claude Codeはこの中で動きます。

普段

マウスで操作

  • アイコンをダブルクリック
  • メニューから「保存」
  • 視覚的でわかりやすい
ターミナル

文字で操作

  • ls でファイル一覧
  • cd フォルダ名 で移動
  • 速い・記録に残る

怖そうに見えますが、実は いつもの操作を「言葉」に置き換えただけ です。本編では細かい使い方は出てきません。「ターミナルで Claude Code が動く」ということだけ覚えてください。

もっと詳しく: Mac と Windows でターミナルはどう違う?

Macでは Terminal.app(アプリケーション > ユーティリティ)、Windowsでは PowerShell または Windows Terminal を使います。Claude Code はどちらでも動きます。Macの方が ls や cd といった伝統的なコマンドが使えるため、解説記事との互換性が高いです。Windowsの場合は WSL (Windows Subsystem for Linux) を入れると、Macと同じ感覚で扱えます。

Node.js と npm って?

Claude Code は Node.js という土台の上で動きます。Node.js を入れると、自動で npm(Node Package Manager)というパッケージ管理ツールも付いてきます。

Node.js / npm / Claude Code の関係
graph LR
  N["Node.js (土台)"] --> P["npm (配布ツール)"]
  P --> C["Claude Code"]
  P --> O["他の開発ツール"]
Node.js を入れる → npm が使える → Claude Code を npm でインストール、という流れ。
  • Node.js を入れる — 公式サイト nodejs.org からダウンロード。LTS版を選べばOK
  • npm が使える状態になる — Node.jsをインストールすると自動で付いてくる
  • npm install で道具を増やす — Claude Codeも npm 経由でインストール

APIキーって?

Claude Code は Anthropic 社のAI(Claude)を呼び出して動きます。「あなたが正規の利用者である」と証明するために、APIキー という長い文字列を使います。パスワードと同じくらい大事。

項目内容
形式sk-ant-... で始まる長い英数字
取得場所Anthropic Console (console.anthropic.com)
料金従量課金 (使った分だけ。1時間試して数十円〜数百円)
取り扱いSNSやコードに貼り付けない。漏れたら即無効化
注意

APIキーは、銀行のキャッシュカードと暗証番号がセットになったようなもの。漏れると他人が課金できてしまいます。スクリーンショットや動画にも映らないように。

Git ってなに?

Git は「ファイルの変更履歴を残すツール」です。GitHub はそのGit用のサービス(クラウド保管庫)です。Claude Code は変更前の状態を Git で保存しておくと、安心して試行錯誤できます。

Git
変更履歴を保存する仕組み
GitHub
Gitの保管庫を置けるWebサービス
commit
変更を保存する操作(セーブ)
push
ローカルの変更をGitHubに送る

本編Ch06でじっくり扱います。今は「Git は変更履歴のセーブ機能」「GitHub はそれを置くオンライン保管庫」とだけ理解しておけば十分です。
