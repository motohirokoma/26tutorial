インストールと初回起動

Node.jsを入れて、Claude Codeをインストールし、初めて「こんにちは」と話しかけるまで。

最初の対話
合計でだいたい30〜60分。途中で詰まったらCh07のトラブルシュートに飛んでください。

Step 1: Node.js を入れる

まず Node.js を入れます。これは Claude Code が動くための土台で、付属の npm も同時に入ります。

  • nodejs.org にアクセスして「LTS」版をダウンロード
  • インストーラを実行(全部「次へ」でOK)
  • ターミナルで node -v と打って v20.x.x 等が出ればOK
# Mac / Linux / Windows 共通の確認コマンド
node -v
# → v20.11.0 (など、バージョンが返ればOK)

npm -v
# → 10.2.4 (など)
もっと詳しく: LTS と Current の違いは?

LTS (Long Term Support) は長期サポート版。安定して動くことが優先されています。Currentは最新機能を試す用。Claude Codeは LTS で問題なく動くので、迷わず LTS を選んでください。

Step 2: Claude Code をインストール

npm 経由で1コマンドで終わります。

npm install -g @anthropic-ai/claude-code

# インストール後、以下で確認
claude --version
# → 1.x.x (など)
-g って何?

グローバルインストール という意味。これでどのフォルダからでも claude コマンドが使えるようになります。

Step 3: APIキーを取得・設定

Claude Code は Anthropic の Claude API を呼び出します。APIキー を1本取得しましょう。

APIキー取得から登録までの流れ
sequenceDiagram
  participant U as あなた
  participant C as Anthropic Console
  participant CLI as Claude Code
  U->>C: アカウント作成 / ログイン
  U->>C: 「Create Key」をクリック
  C-->>U: sk-ant-... (1回だけ表示)
  U->>U: 安全な場所にコピー
  U->>CLI: 環境変数に設定
  CLI-->>U: 認証成功 → 起動
APIキーは表示されるのが「初回1回だけ」。失くしたら新しく作り直せばOK。
  • console.anthropic.com でアカウント作成 (Googleログイン可)
  • 「Billing」で支払い方法を登録 (最低$5から)
  • 「API Keys」 →「Create Key」 → コピー

環境変数として設定する

毎回コピペするのは面倒なので、環境変数 に入れておくのが定番です。

# Mac / Linux (zshの場合)
echo 'export ANTHROPIC_API_KEY="sk-ant-..."' >> ~/.zshrc
source ~/.zshrc

# Windows (PowerShell)
[Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY", "sk-ant-...", "User")
セキュリティ

APIキーは 絶対に公開リポジトリに含めない。ChatGPTに貼り付けるのもNG。漏れたら即 Anthropic Console で「Revoke」して無効化しましょう。

Step 4: 初回起動

専用の作業フォルダを作って、その中で起動するのが定番。

# Macの場合: デスクトップに練習用フォルダを作る
mkdir ~/Desktop/claude-practice
cd ~/Desktop/claude-practice

# Claude Code 起動
claude

起動すると、ターミナルに以下のようなプロンプトが表示されます。

初回起動後のターミナル画面
プロンプト「>」が出たら準備完了。ここに日本語で打ち込めます。

Step 5: 最初の挨拶

記念すべき1回目。なんでも構いませんが、定番は「自己紹介してください」。

> こんにちは。あなたができることを3つ教えてください。

数秒待つと、Claudeが日本語で返事してくれます。返事を見たら、勝ちです。これでもうあなたはClaude Code利用者になりました。

料金感の目安

「使ってると課金されるんでしょ?いくらくらい?」が誰もが気にする点。実測ベースで体感を共有します。

学習1時間あたりの目安課金 (実測)

このチュートリアル全10時間を回しても、概算で$5〜$15(約750円〜2,200円)程度。書籍1冊より安いと考えれば、コスト面の心配はほぼ無用です。

節約のコツ

不要な会話履歴は /clear でリセット。長すぎる場合は /compact で要約してくれます。詳しくはCh03で扱います。

Exercise 1
入門

自己紹介を3つの形式で出させる

初回起動後、Claudeに同じ質問を「形式を変えて」3回繰り返してみましょう。応答の柔軟性を体感する練習です。

  1. claude を起動
  2. 「自分の自己紹介を箇条書きで3行」と頼む
  3. 「次は俳句で」と頼む
  4. 「最後は子供向けの優しい言葉で」と頼む
模範解答を見る

たとえばこんなふうに返ってきます。形式が変わっても内容は同じテーマ(=Claude Code)を保てるのがエージェント型の強み。

【箇条書き】
- ターミナルで動くAI開発アシスタント
- 日本語で頼める
- ファイルを直接編集できる

【俳句】
コードかき
ターミナル住む
クロード哉

【子供向け】
こんにちは!ぼくはクロード。
キミのパソコンの中のお手伝いさんで、
「こんなものほしい」と話しかけると
それを作ってくれるよ。
Exercise 2
応用

練習用フォルダに README.md を作らせる

初めて「Claudeにファイルを作らせる」体験をします。ファイル操作の最初の一歩です。

  1. ~/Desktop/claude-practice 内で claude 起動中であることを確認
  2. 「このフォルダに README.md を作って、私の学習記録ページであることを書いて」と依頼
  3. Claudeが提案する内容を確認して「OK」
  4. ターミナルを別タブで開いて ls で README.md ができていることを確認
模範解答を見る

Claudeはこんな感じの README を提案してきます。「これでよろしいですか?」と聞かれるので、Enterで承認。
