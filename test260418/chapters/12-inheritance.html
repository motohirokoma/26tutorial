相続の基本知識

大学生〜新卒でも「親と話すとき」「親が亡くなったとき」に迷わない最小知識

所要時間: 30分 Phase: Advanced 難易度: 中級

なぜ学生・新卒が相続を学ぶのか

「相続って高齢者の話でしょ?」と思うかもしれない。しかし相続は 親が元気なうちから始まっている。生前贈与・遺言書・保険の受取人設定など、親と会話しておくべきテーマは多い。いざ相続が発生したとき、手続きには期限があり、知らないと損をする。

Tips

相続は「亡くなった瞬間」に自動で発生する。その日から 3ヶ月以内に相続放棄するか判断、10ヶ月以内に相続税を申告 という厳しいスケジュールが回り始める。

「被相続人」になるのは親・祖父母だけとは限らない。独身の兄姉が亡くなって、自分が相続人になるケースもある。

法定相続人と法定相続分

遺言書がない場合、民法が定めた法定相続人が、法定相続分で財産を分ける。配偶者は常に相続人となり、血族は 第1順位: 子 → 第2順位: 親 → 第3順位: 兄弟姉妹 の順で1順位だけが相続人になる。

家族構成別の法定相続分
配偶者がいる場合、同順位の血族間では均等に分ける。子・親・兄弟の順位が上なら下は相続人にならない。
もっと詳しく: 代襲相続 / 養子 / 内縁のパートナー

代襲相続: 本来の相続人(例えば子)が先に亡くなっていた場合、その子(孫)が代わって相続する仕組み。兄弟姉妹が相続人の場合は 甥・姪まで が代襲の上限で、再代襲は起きない。

養子: 法律上は実子と同じ扱いで相続人になる。ただし相続税の基礎控除を計算する時の「法定相続人の数」は、実子がいれば養子は1人まで、実子がいなければ2人までとカウント制限がある。

内縁のパートナー: 婚姻届を出していない場合、相続権はゼロ。遺言書で明示するか、生命保険の受取人に指定する、生前贈与するなどの対策が必要。

相続税の基礎控除 — 実は91%の人が払わない

相続税には大きな 基礎控除 がある。この枠を超えた部分にだけ税金がかかり、超えなければ相続税はゼロ・申告も不要。

基礎控除額 = 3,000万円 + 600万円 × 法定相続人の数 — 2015年改正後の相続税法
基礎控除の計算例(配偶者+子2人)
3000万 + 600万 × 3人 = 4,800万円までは非課税。

日本人の平均的な相続財産(現預金+持ち家)は多くの場合この枠に収まる。だから 91%の相続では相続税が発生しない。「相続=税金がかかる」はよくある誤解。

注意

ただし都市部で持ち家があり、預貯金も貯まっている世代では基礎控除を超えるケースも多い。東京23区の戸建て1軒で土地評価額が5,000万を超えることは珍しくない。

配偶者の特例

配偶者の税額軽減: 配偶者が相続する分は 1億6,000万円まで、または法定相続分まで は相続税がかからない。実質的に夫婦間の相続ではほぼ税金ゼロ。ただし次の代(子)への相続時にまとめて課税されるので、配偶者に寄せすぎる節税は要注意。

遺言書 — 3種類の違いを押さえる

遺言書があれば、法定相続分と異なる配分も可能(ただし 遺留分 は侵害できない)。普通方式の遺言書には3種類ある。

種類作成方法費用証人検認おすすめ度
自筆証書遺言 本人が全文手書き(財産目録のみPC可) 0円 不要 必要(法務局保管なら不要) 手軽だが無効リスク
公正証書遺言 公証役場で公証人が作成 数万〜数十万円 2人必要 不要 確実・最も安全
秘密証書遺言 本人が作成し公証役場で封印 1.1万円程度 2人必要 必要 ほぼ使われない
もっと詳しく: 2020年の自筆証書遺言ルール改正

2020年7月から 自筆証書遺言書保管制度 が始まった。法務局に有料(3,900円)で保管してもらえば、

  • 紛失・改ざんのリスクが消える
  • 家庭裁判所での 検認 手続きが不要になる
  • 相続人に死亡通知も届く(希望すれば)

また2019年からは 財産目録のみワープロ・PC作成可になり、自筆の負担が大幅に減った。

Tips

自筆証書遺言は「日付・氏名・押印・全文自筆」のどれか1つでも欠けると無効になる。不安なら公正証書遺言、または法務局保管制度付きの自筆証書遺言が安全。

生前贈与 — 元気なうちから渡す

相続を待たずに、生きている間に財産を渡すのが 生前贈与。贈与税の課税方式は2つから選べる。

暦年贈与

年110万円まで非課税

  • 毎年1月1日〜12月31日で計算
  • もらう側1人あたり年110万円まで無税
  • 10年続ければ1,100万円を非課税で移転
  • 相続開始前7年以内の贈与は相続財産に加算(2024年改正で3→7年に拡大)
相続時精算課税

累計2,500万円まで贈与時無税

  • 60歳以上の親・祖父母 → 18歳以上の子・孫
  • 贈与時は累計2,500万まで非課税、超過分は一律20%
  • 相続発生時に贈与額を加算して精算課税
  • 2024年から年110万の基礎控除が追加(併用可)
実務アドバイス

暦年贈与を使う時は 贈与契約書を毎年作る / 銀行振込で記録を残す / もらう側の口座を本人が管理する の3点を守る。親がこっそり子名義の口座に振り込んでいるだけだと「名義預金」とみなされ、相続時に親の財産として課税される。

相続放棄 — 借金は相続しない選択

相続は プラスの財産だけでなく借金もセットで引き継ぐ。親が多額のローンや事業の借金を残していた場合、放っておくと子が返済義務を負う。これを避けるのが 相続放棄。

何もしないと「単純承認」扱いで借金も引き継ぐ。判断できない場合は期間伸長の申請も可能。
警告

相続財産を 1円でも使うと「単純承認」した とみなされ、もう相続放棄できなくなる。親の預金を葬儀代に充てる、遺品を処分するなども該当しうる。判断前に何かに手を付けるのは避けること。

相続放棄すると次の順位に相続権が移る。例えば子が全員放棄すれば親、親もいなければ兄弟姉妹に借金が回る。放棄する場合は 他の親族にも連絡して連鎖的に対応 するのが望ましい。

相続手続きのタイムライン

いざ相続が発生したとき、10ヶ月で走らせるマラソン。期限を知っていれば慌てずに済む。

7日以内
死亡届の提出
市区町村役所へ。葬儀社が代行することも多い。火葬許可証も同時に取得。
〜1ヶ月
遺言書の有無確認・相続人の調査
被相続人の戸籍を出生まで遡って取り寄せ、相続人を特定。自筆証書遺言があれば家庭裁判所で検認。
〜3ヶ月
相続財産の調査 + 放棄の判断
預貯金・不動産・有価証券・借金をリストアップ。マイナスが大きければ相続放棄を家裁に申述。
〜4ヶ月
準確定申告
被相続人の亡くなった年の1月1日〜死亡日までの所得税を、相続人が代わりに申告・納税。
〜10ヶ月
遺産分割協議 + 相続税申告・納税
相続人全員で分け方を協議し、遺産分割協議書 を作成。基礎控除を超えるなら税務署に申告。
Tips

相続人が基礎控除を超える財産を引き継ぐなら、税理士への相談を早めに。報酬は相続財産の0.5〜1%が相場だが、特例適用や評価減で節税できる額のほうが大きいことが多い。

演習問題

Exercise 1
入門

自分の家族での法定相続分を計算する

以下のAさんの家族構成で、もしAさんが亡くなったら誰がいくら相続するか、法定相続分で計算してみましょう。

  1. Aさん(被相続人)の相続財産: 6,000万円
  2. 家族: 配偶者、長男、長女の3人
  3. それぞれの相続分(金額)を求める
  4. おまけ: もし長男が先に亡くなっていて長男に子(Aさんから見た孫)が1人いた場合は?
模範解答を見る

基本ケース(配偶者+子2人)

  • 配偶者: 1/2 → 3,000万円
  • 長男: 1/4 → 1,500万円
  • 長女: 1/4 → 1,500万円

配偶者は常に1/2、残り1/2を子2人で均等に分ける。

長男が先に亡くなり孫がいる場合(代襲相続)

  • 配偶者: 1/2 → 3,000万円
  • 孫(長男の代襲): 1/4 → 1,500万円
  • 長女: 1/4 → 1,500万円

孫が長男の枠をそっくり引き継ぐ。孫が複数なら1/4をさらに均等分割。

Exercise 2
応用

基礎控除を超えるかシミュレーション

Bさんの家族の状況で、相続税がかかるかどうかを判定しましょう。

  1. Bさん(父)の相続財産: 自宅評価額 3,800万円、預貯金 2,200万円、生命保険金 1,500万円(受取人: 母)
  2. 家族: 配偶者(母)、子3人
  3. 基礎控除額を計算
  4. 課税財産を計算(生命保険金の非課税枠に注意: 500万円 × 法定相続人数)
  5. 相続税がかかるかどうか、かかるなら概算いくらか
模範解答を見る

1. 基礎控除額

3,000万 + 600万 × 4人(配偶者+子3人) = 5,400万円

2. 生命保険金の非課税枠

500万 × 4人 = 2,000万円 が非課税
生命保険金 1,500万 − 非課税枠 2,000万 = 0円(全額非課税)

3. 課税財産の合計

自宅 3,800万 + 預貯金 2,200万 + 保険 0 = 6,000万円

4. 基礎控除後の課税対象

6,000万 − 基礎控除 5,400万 = 600万円

5. 相続税額の概算

法定相続分で分けた想定課税額を計算し、各人の税率を適用:

  • 配偶者法定相続分: 600万 × 1/2 = 300万 → 10%適用で30万円(ただし配偶者税額軽減で実質0)
  • 子各人: 600万 × 1/6 ≒ 100万 × 3人 → それぞれ10%で10万 × 3 = 30万円

結論: 相続税は家族合計で概算30万円程度。ただし小規模宅地の特例(自宅土地を80%評価減)が使えれば、基礎控除以下に収まり税額ゼロになる可能性が高い。

このように「ギリギリ超える」ケースは特例適用で実質非課税になることが多い。都市部の持ち家がある家族こそ、早めに税理士に相談する価値がある。
