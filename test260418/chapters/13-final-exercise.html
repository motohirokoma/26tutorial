最終演習問題：あなたのライフプラン

25歳から65歳までの人生を、キャッシュフロー表で設計してみる

所要時間: 45分 Phase: Advanced 難易度: 応用

この演習で統合する12章の学び

いよいよ最終章です。Ch01からCh12まで、給与・税金・年金・家計・投資・保険・住宅・相続と、お金に関する幅広い知識を積み上げてきました。ここでは、それらを ひとつの表に落とし込みます。目標は、キャッシュフロー表（CF表）で25歳から65歳までの自分の人生を数字で描いてみること。

「学んだけれど、どう使うのかピンと来ない」ーーその霧を晴らす演習です。紙に書いてもExcelで作っても構いません。とにかく手を動かして数字を置いてみること。それだけで、漠然としていた「老後が不安」「家を買えるのか」という感覚が、根拠のある数字に変わります。

Tips

完璧な未来予測を作る必要はありません。前提条件は 一度固定して、その上で40年分の数字を通しで並べる。それだけで「不足するのはどのタイミングか」「貯蓄率をいつ上げるべきか」が見えてきます。

1枚のCF表の中に、学んできたすべての知識が並ぶ。

問題: 25歳→65歳の40年CF表を作る

あなたは25歳の新卒社員です。以下の前提条件のもとで、65歳までのキャッシュフロー表を作り、各ライフステージで金融資産がいくらになっているかを描いてください。

前提条件 (モデルケース)

項目内容
年齢・属性25歳・新卒会社員 (男性モデル、女性・独身ケースも応用で扱う)
初任給月25万円・年収 約350万円 (手取り約280万)
昇給率年1.5% (ただし45歳以降は横ばい、50歳以降は役職定年で▲10%)
結婚30歳・配偶者は同年齢・年収400万
出産35歳・子ども1人
住宅購入35歳・物件3,000万・頭金500万・35年固定1.5%
配偶者の働き方出産後は時短→子ども15歳で正社員復帰 (年収350万想定)
定年65歳 (60歳で役職定年、65歳再雇用終了)
公的年金65歳から夫婦で月22万 (Ch03の想定)

求めるアウトプット

  • CF表 (5年刻み9ポイント) — 25/30/35/40/45/50/55/60/65歳それぞれの収入・支出・貯蓄・金融資産残高
  • 金融資産の推移グラフ — 40年分の累積。住宅購入での凹み、NISA/iDeCoの複利効果を視覚化
  • 65歳時点の資産配分 — 預貯金・NISA・iDeCo・持ち家(評価額)の割合
  • ボトルネック分析 — 一番お金が苦しくなる年齢と、その対策を一言
Note

CF表は「正確な未来予測」ではなく「前提を置いたときの見取り図」です。物価上昇率・昇給率・運用利回りを変えれば結果は揺れます。大事なのは、どの変数が効くかを体感することです。

段階ヒント (行き詰まったら開く)

一気に全部見ると「答え写し」になります。まずは自分で試算して、詰まった項目からひとつずつ開いてください。

もっと詳しく: ヒント1 — 収入の推移 (Ch02/Ch03の復習)

手取り=額面×(1-税率)で粗く、本モデルでは 額面の約80%を手取りとして扱います (Ch02)。

  • 25歳: 額面350万 → 手取り約280万
  • 30歳: 年1.5%昇給で約400万 → 手取り約320万 (本人のみ)
  • 35歳: 約430万 → 手取り約345万 + 配偶者時短300万
  • 45歳: 約500万でピーク → 手取り約400万 + 配偶者350万
  • 55歳: 役職定年で▲10% → 手取り約360万 + 配偶者350万
  • 60-65歳: 再雇用で年収300万想定 → 手取り約240万
  • 65歳〜: 公的年金のみ夫婦で年264万 (月22万)

Ch11で学んだ所得控除 (iDeCo掛金・生命保険料・医療費・ふるさと納税) で実質手取りは5〜10万/年プラスになる点も忘れずに。

もっと詳しく: ヒント2 — 支出の推移 (Ch04/Ch08/Ch09/Ch10)

支出は ライフステージで段階的に膨らむのが特徴です。

ステージ年間支出目安主な内訳
25-29 単身約250万家賃8万×12 + 食費生活費8万×12 + 娯楽交際20万
30-34 DINKS約400万家賃12万 + 食費生活15万 + 保険 + 積立
35-44 住宅+子0-9歳約480万ローン92万 + 生活290万 + 教育50万 + 保険50万
45-54 教育費ピーク約550万ローン92万 + 生活300万 + 教育130万 + 保険30万
55-64 子独立後約400万ローン92万 (64歳で完済) + 生活280万 + 保険軽量化30万
65- 老後約300万生活280万 + 医療介護予備20万 (ローンなし)

保険料はCh08/Ch09の考え方で「必要保障額 (遺族の生活費-遺族年金-貯蓄)」から逆算。子独立後は大幅に下げられます。

もっと詳しく: ヒント3 — 貯蓄率の目安 (Ch04)

Ch04で学んだ貯蓄率の段階的引き上げがそのまま使えます。

新卒〜20代後半
10% (月2.3万)
結婚〜住宅購入前
20% (月10万/世帯)
子育て期
15% (ローン後)
子独立〜定年
30%超 (月15万)

「貯蓄」には 預貯金+NISA+iDeCoすべてを含めて考えること。銀行預金だけで貯蓄率20%だと、インフレで目減りします (Ch05)。

もっと詳しく: ヒント4 — NISA/iDeCo配分と複利 (Ch06/Ch07)

モデルケースでは 月3万円×40年をNISA/iDeCoで積み立てます。年利5% (全世界株インデックス想定) で複利運用すると:

  • 元本: 3万×12×40 = 1,440万円
  • 運用後: 約4,460万円 (約3倍)
  • 運用益: 約3,020万円が 非課税で手元に残る

比較として、同額を普通預金 (年0.02%) に置くと40年後 約1,445万円。3,000万円の差が40年の時間軸で生まれます (Ch06の複利・72の法則)。

Tips

iDeCoは月2.3万 (会社員で企業年金なしの枠) まで全額所得控除。課税所得300万円なら年約5.5万円の節税。これもCF表の「隠れた手取り増加」として計上を。

もっと詳しく: ヒント5 — 住宅購入のタイミングと返済額 (Ch10)

Ch10の「返済比率25%以下」「年収の5-7倍以内」がそのまま制約になります。

  • 35歳時の世帯年収: 約750万 → 借入上限目安 3,750〜5,250万
  • 今回は 2,500万借入 (3,000万-頭金500万) で 安全圏
  • 月返済 約7.7万 (年92万) は世帯手取り約600万の 15%で健全
  • ローン完済は70歳 → 65歳時点で残債約520万、退職金で完済想定

頭金500万を貯めるために、25-34歳の10年で 年50万×10 = 500万の預貯金シフトが必要。その期間のNISAは月1万に抑え、キャッシュ優先で積み上げるのがコツ。

考え方の解説

この解答の肝は 3つの切り替え点です。

  1. 30歳の世帯化で貯蓄率を一気に20%へ — 単身のぬるい10%から切り替えないと頭金500万が貯まらない。
  2. 35歳で500万を一気に投入しても金融資産は途切れない — 配偶者収入+共働きで翌年から再び積み上げ可能。
  3. 55→60歳の5年で1,000万を積み上げる — 教育費が消え、子が独立し、役職定年前の「貯めどき」を逃さない。
Warning

このモデルは 配偶者の就労継続を前提にしています。配偶者が専業主婦/主夫になる場合、世帯手取りが年300〜400万下がり、65歳時の金融資産は 約2,500万まで縮みます。その時は住宅予算を2,500万に抑えるか、NISA積立を月2万に下げる調整が必要。

最大のボトルネック

  • 年間貯蓄が40万円まで落ちる
  • ローン・教育費・保険が三重で重なる
  • 想定外 (親の介護等) で赤字化リスク
処方箋

  • 学資はNISA+児童手当で月1.5万×18年 (Ch07)
  • 保険は収入保障型へ (Ch08) で月1万円圧縮

応用課題: 別ケースで比較する

モデルケースが解けたら、前提を変えると65歳時点の資産がどう動くかを比べてみましょう。

応用 1
応用

DINKS (夫婦共働き・子なし) の場合

モデルケースから「出産・教育費」を抜き、代わりに「住宅を35歳で4,000万のマンション購入」に差し替えて65歳時点の金融資産を試算してください。配偶者は30歳からフルタイム継続 (年収400万→昇給で500万) とします。

  1. 35-54歳の年間貯蓄率は何%まで上げられるか?
  2. 教育費ゼロで浮いた累計は概算いくらか?
  3. 住宅予算を上げた場合のローン返済額と65歳完済可否を確認
模範解答を見る

教育費ゼロで浮くのは 18-22歳×150万 ≒ 2,700万相当。うち半分をNISA/iDeCo積立に回すと、65歳時点の金融資産は 約7,800万に達します (モデルケース+2,100万)。

  • 年間貯蓄率は35-54歳で 30%超が現実的
  • 住宅4,000万・頭金800万・35年固定1.5%で月返済9.8万 (世帯手取り700万の17%で安全)
  • 65歳時の住宅ローン残債は約430万 → 退職金で完済可能
Insight

DINKSの金銭的余裕は大きいですが、65歳以降の介護リスクは子あり家庭より高い。その分 介護・医療の予備費を多め (+500万) に確保する設計が推奨されます (Ch09/Ch12)。

応用 2
応用

独身継続 / 女性新卒モデル

25歳女性・独身継続・年収320万スタート・年1.2%昇給・結婚/出産なし・35歳でマンション2,500万購入、というケースで65歳時の金融資産を試算してください。

  1. 世帯収入が1人分で伸び悩む中、どうNISA/iDeCoを積み立てるか
  2. 独身ゆえに必要な保険 (就業不能・医療) の取捨
  3. 老後の単身生活費 月18万×30年 = 6,480万をどこから賄うか
模範解答を見る

独身モデルは 収入は少ないが支出も少ないのが特徴。

  • 手取りピーク (45歳) で約380万・年間支出270万 → 貯蓄率30%維持可能
  • NISA月3万+iDeCo月2万で40年、利回り5%なら約7,440万に到達
  • 65歳金融資産は持ち家 (評価1,200万)+現金500万+運用7,440万 ≒ 9,140万
  • 生命保険は不要、代わりに 就業不能保険 (Ch08) と 医療保険の手厚め加入 (Ch09) を優先

結論: 独身モデルは「投資の継続力」が勝負。年金 (月13万)+運用取り崩し (年3%=約220万) で老後月約30万の生活が成立します。

Tips

女性モデルの場合、平均寿命が87歳と長いため、95歳までの資金計画で考えるのが保守的。取り崩しは年3〜3.5%に抑え、iDeCoの受給は「年金方式」で長期分散するのが有効です。
