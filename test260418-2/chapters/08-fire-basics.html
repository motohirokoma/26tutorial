焚き火入門

火起こし・薪の扱い・焚き火台・安全管理

所要時間: 30分 Phase: Core 難易度: 初級
この章のゴール

「焚き火台の選び方」「薪の選び方」「火起こしの手順」「使用中の安全管理」「後始末」の5つを一気通貫で理解する。読み終わったら自宅の庭やベランダで練習できるレベルに到達し、初めてのソロキャンプで迷わず火を起こせる状態を目指します。

焚き火の4つの役割

ソロキャンプで「焚き火だけが目的」という人も多いほど、焚き火は中心的な体験です。暖・調理・光・心理の4つの役割を一度に担える、キャンプ場の万能ツールと言えます。

焚き火セラピー

燃焼の三要素を知ることは、火起こしの成功だけでなく安全に消火するためにも必須です。下の三角形を頭に焼き付けてください。

火の三要素(燃焼の三角形)
3つのうち1つでも断てば火は消える。消火時は「酸素を断つ=水/蓋」「燃料を断つ=薪を足さない」の合わせ技が基本。
Tips

ソロキャンプの焚き火は「手元で制御できる小さな火」で十分。豪快な大焚き火は映像映えしますが、薪の消費・火の粉・鎮火時間すべてが厳しくなります。「両手で隠せるサイズの炎」を目安にしましょう。

焚き火台の種類と選び方

いまの日本のキャンプ場はほぼ全て直火禁止。地面を焦がさないために焚き火台の持参が必須です。ソロで主流の4タイプを比較しましょう。

  • DOD 秘密のグリルちゃん — 大型で調理兼用、ファミリー寄りだがソロでも余裕ある火床

  • ファイアーボックス — 米国製、スティック投入口がある箱型

選び方の優先順位は「①徒歩か車か(重量)」「②調理するか(上面形状)」「③薪の入手形態(40cm?20cm?)」の順で絞ると迷いません。

Tips: 焚き火シートを忘れずに

焚き火台の下にはスパッタシート(耐熱シート)を必ず敷きます。火の粉が芝や地面を焦がすのを防ぐため、多くのキャンプ場でルール化されています。2,000〜3,000円で入手可能。

薪の種類と見極め方

薪は大きく広葉樹と針葉樹に分かれます。用途に応じて使い分けるのがプロの第一歩です。

種類代表樹種火付き火持ちヤニ/煙価格用途
広葉樹ナラ / クヌギ / カシ / サクラ△ やや悪い◎ 長い少ないやや高めメインの焚き火・長時間調理
針葉樹スギ / ヒノキ / マツ◎ 良い△ 短い多め(マツは特に)安い焚きつけ・火起こし初動
混合両方のミックス○○標準標準一般的な構成

ベストな組み合わせは「針葉樹で火を起こして、広葉樹で本番を楽しむ」。最初から広葉樹だけだと着火に苦労します。

薪の乾燥度チェック
乾燥薪は見た目に割れ目があり、持つと軽く、叩くと「カン」と乾いた音がする。
Tips: 現地購入が一番楽

多くのキャンプ場は薪(1束500〜800円、約5kg)を販売しています。自宅から運ぶと車が汚れる&虫を持ち込むリスクがあるので、現地購入が基本。1泊なら2束、冬なら3束が目安。

もっと詳しく: 薪通販と薪棚

本格的に焚き火を楽しむなら、乾燥度の高い薪を通販で購入するのも一つ。「薪の専門店 マルトモ」「薪ストーブ本舗」「ファイヤーサイド」などが有名です。含水率20%以下を目安に選ぶと失敗しません。

家の軒下に薪棚を作って自分で乾燥させる「薪割り生活」に入る人も。生木を1年以上風通しの良い場所に置くと、含水率が30%前後まで下がり最高の薪になります。

火起こしの5ステップ

火は「小さい順に育てる」のが鉄則。火口(ほくち)→焚き付け→小割り→中割り→本薪の順に、段階的に大きくしていきます。

1
火口を置く
麻紐・着火剤・松ぼっくり
2
焚き付け
細い針葉樹(鉛筆太)
3
小割り
親指太の薪を三角に
4
中割り
握り太の薪を井桁に
5
本薪
広葉樹を1本ずつ投入
火起こしのフロー(つまずき対処付き)
graph TD
  Start([焚き火開始]) --> S1["火口を焚き火台中央に配置"]
  S1 --> S2["焚きつけ(細針葉樹)を三角に立て掛け"]
  S2 --> S3["着火: ライター/ファイヤースターター"]
  S3 --> Q1{"炎が安定したか?"}
  Q1 -- "Yes" --> S4["小割り薪を追加"]
  Q1 -- "No" --> F1["空気を送る / 火口追加"]
  F1 --> S3
  S4 --> S5["火が落ち着いたら中割り投入"]
  S5 --> S6["熾火になったら本薪(広葉樹)"]
  S6 --> End([安定した焚き火])
「炎が消えそう→焦って大きい薪を足す」は最悪。空気・火口を足して小さく育て直すのが正解。

着火道具の選択肢

  • ライター / ターボライター — 初心者はまずこれ。100円ライターでも十分、風に強いターボ式(SOTO スライドガストーチ等)があると尚良し。
  • ファイヤースターター — マグネシウム棒を削って火花を飛ばす。ブッシュクラフト感が出るが練習必須。
  • 着火剤 — 固形タイプ(ロゴス・Coleman)が最強。牛乳パック、ワセリン染込ませた綿も代用可。
  • 火吹き棒 — 空気をピンポイントで送る。口で息を吹くより断然楽で煙も目に入らない。
Tips: フェザースティック

針葉樹の角をバトニングで割り、ナイフで薄く削って毛羽立たせたフェザースティックを作ると、着火剤なしでもマッチ1本で火が着きます。できるようになると雨天時の保険になります。

使用中の安全管理 — 風向き・距離・服装

焚き火が安定したら楽しい時間。でも事故の9割は油断した瞬間に起きます。以下のルールを頭に入れて。

焚き火の配置と危険距離
テントと焚き火台は風下で最低3m。化繊テントは火の粉1個で穴が開くので「煙が顔に来る向き=配置ミス」と覚える。
NG行動何が起きるか正しいやり方
風上にテントを置く煙が充満・火の粉で穴焚き火台は風下、距離3m以上
化繊フリースで近づく火花1個で穴 / 最悪燃えるコットン系の焚き火ウェア着用
ハイチェアから見下ろす重心高く転倒・顔に火の粉ロースタイルチェアで目線を合わせる
薪を大量に一気投入火柱・火の粉飛散1本ずつ、状態を見ながら
離席して放置強風で飛び火・火災片時も目を離さない
アルコールで酩酊判断力低下で事故焚き火中は控えめに
警告: 強風時は中止する勇気を

風速5m/s以上の予報が出ている、または現地で小枝が揺れるような風が吹いているときは、焚き火を諦める判断が必要です。飛び火は山火事の原因No.1。ガスバーナーに切り替えて調理だけ済ませましょう。

もっと詳しく: 焚き火ウェアと道具

焚き火を頻繁にやるようになったら、専用ウェアの導入を検討してください。

  • グリップスワニー ファイヤープルーフパンツ — コットン製、火の粉に強い
  • ロゴス たき火台トング — 薪を掴む、40cm以上の長さが欲しい
  • 革手袋(牛革/鹿革) — WORKMANで1,000円程度から
  • 火吹き棒 — TOKYO CRAFTS KAKUBOU等、携帯性重視なら伸縮式
  • ナタ(鉈) — バトニング用、モーラナイフで代用も可

後始末 — 完全消火と灰の処理

焚き火で最も責任が問われるのが後始末。「まだ燃え残ってるけど大丈夫だろう」は通用しません。次の利用者への礼儀でもあります。

鎮火までの時間を逆算する

本薪(広葉樹)を最後に投入するのは、就寝/撤収の90〜120分前を目安に。以降は新しい薪を足さず、自然に燃え尽きさせます。

-120分
最後の本薪投入
以降は足さない。火力を細らせていく期間に入る。
-60分
熾火(おきび)の状態に
炎がなくなり、赤く発光する状態。まだ高温。
-30分
灰が大半に
熾火が白い灰で覆われてくる。まだ触らない。
0分
水で完全消火 → 火消し壺へ
水をかけて蒸気が出なくなるまで。手で触れる温度になってから灰捨て場 or 火消し壺へ。
重要: 土に埋める/草むらに捨てるは絶対NG

熾火は土中で数時間〜1日くすぶり、地中火災や山火事の原因になります。必ずキャンプ場の灰捨て場に入れるか、密閉式の火消し壺に回収して持ち帰ってください。

禁止エリア・直火禁止の確認

  • 国立公園の特別保護地区 — 焚き火が完全禁止の場合あり。環境省サイトで事前確認。
  • 一部のキャンプ場で焚き火禁止 — 芝保護のため、焚き火台ですら不可の場所もある。予約時に必ず確認。
  • 乾燥注意報・火災警報発令中 — 自治体の判断で一時的に禁止になることも。天気予報と併せてチェック。
  • 焚き火OK確認済みサイト — なっぷ等の予約サイトで「焚き火可」絞り込みが使える。
Exercise 1
入門

自宅で火起こし練習セットを揃える

初キャンプ前に、自宅の庭・ベランダ(集合住宅は不可)・公園(許可要)で火起こしの練習をしておくと、本番の不安が激減します。今回は練習セット一式をリスト化してみましょう。

  1. 焚き火台(なければ空き缶BBQコンロでも可)と耐熱シートをそろえる
  2. ホームセンターで針葉樹の薪1束(500円程度)を購入
  3. 着火道具3種(ライター・ファイヤースターター・着火剤)を用意
  4. 火口(麻紐・松ぼっくり・牛乳パック)を1つ以上準備
  5. バケツ1杯の水を必ず横に置く
模範解答を見る

合計予算2,000〜5,000円で揃います。おすすめ構成:

  • 焚き火台: DOD めちゃもえファイヤー(4,000円)or 笑's B-6君(3,000円)
  • 薪: ホームセンター(カインズ・コメリ)で針葉樹1束 400〜600円
  • 着火道具: SOTO スライドガストーチ(2,000円)が長持ちかつ風に強い
  • 火口: 麻紐1束(100円)をほぐして使う
  • 安全: バケツ・革手袋・火消し壺(2,000円)

練習のコツ:「着火→10秒以内に炎が焚きつけに移る」「空気を送らずに3分維持できる」を目標に5回繰り返す。これで現地での火起こしは8割成功します。

Exercise 2
応用

自分のスタイルに合う焚き火台を比較選定する

本章で紹介した代表4モデル(ピコグリル・ソロストーブ・ユニフレーム・スノーピーク)から、自分の想定するキャンプスタイルに最適な1台を選定し、その理由を言語化しましょう。

  1. 自分のキャンプ移動手段(車/バイク/徒歩)と荷物量の上限を書き出す
  2. 焚き火の用途(眺める専用/調理メイン/煙嫌い/一生モノ欲しい)をはっきりさせる
  3. 上の比較表と compare-cards を使って、2〜3台に絞る
  4. 予算(5,000〜30,000円)と用途のバランスで最終候補を決める
  5. 2万円以上の高価モデルは5年間の使用頻度でコスト妥当性を検討
模範解答を見る

3つのモデルケース:

  • 車キャン月2回・ファミリー兼用・料理重視の30代 → ユニフレーム ファイアグリル(7,700円)。BBQ網・ダッチオーブン両対応で10年使える。
  • バイクソロ月1・眺める専用・煙嫌いの20代 → ソロストーブ レンジャー(22,000円)。二次燃焼の炎は唯一無二。薪を20cm以下にカット前提なので現地調達先確認必須。
  • 徒歩/バイクパッキング・軽量命・オールラウンダーの40代 → ピコグリル398(14,000円)。365gでUL装備と両立、本物の40cm薪を載せられて調理もいける。

決定のフレーム: 「重量 × 用途 × 予算/年数」で採点し、総合点が最も高いものを選ぶ。どれも良品なので「選択肢として正解」はありません。自分のスタイルとの合致度が全てです。
