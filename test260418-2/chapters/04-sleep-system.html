寝袋・マットの選び方

快適温度/化繊vsダウン/R値

所要時間: 30分 Phase: Basics 難易度: 初級
この章のゴール

「夜の寒さで眠れない」はソロキャンプ最大の失敗パターン。寝袋の形状・保温材・温度表記、マットのR値、そしてこの2つを支える三位一体(寝袋+マット+テント)の考え方を理解すれば、行く季節・場所に合わせて迷わず装備を組めるようになります。

まず寝袋の3タイプを知る

寝袋(シュラフ)には大きく3つの形状があります。見た目の違い=保温効率と快適さのトレードオフです。

寝袋3タイプの形状比較
左から保温重視→快適重視→軽量重視。背中側に重量を持つ「キルト」はUL志向の選択肢。

それぞれの特徴

  • マミー型 — ミイラ(mummy)の名の通り、身体のラインに沿ってぴったり。デッドスペースが少なく保温効率最強。ソロキャンプの定番。
  • 封筒型 — 長方形で中がゆったり。家の布団に近い感覚で寝返りが打てる。夏キャンプ・車移動派・女性に人気だが嵩張る。
  • キルト型(クイルト) — 背中側の綿を省略した"布団型"。マットの断熱力に依存する前提の UL仕様。500g台〜で超軽量。
もっと詳しく: なぜキルトで背中を省略するのか

寝袋の中綿は潰れると保温力がゼロに近づく。仰向けで寝ると背中側の綿は体重で圧縮されてロフト(膨らみ)を失い、保温にほとんど寄与しない。「どうせ潰れるなら最初から無くして軽くしよう」というのがキルトの発想。代わりにマット側でしっかり断熱する必要があるので、R値の高いマットとセットで使う前提の装備です。

保温材: ダウン vs 化繊

寝袋の性能を左右する最大の要素は中に詰まっている保温材。大きくダウンと化繊(化学繊維)の2種類です。

ダウン

軽量・コンパクト・高温効率

  • 同じ保温力で化繊の半分以下の重量
  • 圧縮すると小さく収まる(1L台も可能)
  • 温度レンジが広く、冬山までカバー
  • 濡れると一気に保温力が落ちる
  • 価格が高い(3〜8万円)
化繊

濡れに強い・安い・重い

  • 濡れても保温力が落ちにくい
  • 洗濯機で丸洗いできる
  • 価格が安い(5千〜1.5万円)
  • ダウン比で1.5〜2倍重く嵩張る
  • 長期保管で綿がへたる
Tips: フィルパワー(FP)

ダウンの品質指標。FPが高いほど少量で暖かい。エントリーは600FP、山岳モデルは800〜1000FP。同じ快適温度でもFP800なら総重量が3割軽くなるイメージ。

数字で見る「暖かさあたり重量」

快適温度0℃クラスの総重量比較(目安)
同じ"0℃対応"でもFP800ダウンは化繊の半分以下で済む。軽量化=快適性の差。
もっと詳しく: ダウンと化繊のハイブリッド

最近は「ダウンの軽さ+化繊の耐水性」を目指した撥水ダウンや、胸と背中だけ化繊にしたハイブリッドモデルも出ている。ただし初心者は最初から高機能を狙わず、まずは600-800FPのシンプルなダウンか化繊の封筒型で十分。ダウンは手入れ(乾燥・保管)が必要で、圧縮したまま放置するとロフトが戻らなくなります。

温度表記を正しく読む(コンフォート/リミット/エクストリーム)

寝袋のスペック表には3つの温度が書かれています。国際規格EN13537(ISO23537)で定義されており、メーカー横断で比較できる数少ない指標です。

表記別名意味読み方
コンフォートComfort / 快適温度標準的な女性が丸まらず快適に寝られる下限購入時はこれを基準にする
リミットLimit / 下限温度標準的な男性が丸まって寝れば8時間耐えられる下限参考値、ここまで使わない想定で
エクストリームExtreme / 限界温度生存可能なだけ。凍死はしないが眠れない絶対に狙って買わない
重要: 現地気温より5℃低いコンフォート温度を選ぶ

外気温10℃の現地なら、コンフォート5℃の寝袋を選ぶ。疲労・空腹・風・湿気で体感温度は必ず下がるので、スペックギリギリは失敗の元。特に女性・冷え性の方は10℃のマージンを取るくらいが安全です。

月別・エリア別の推奨コンフォート温度
首都圏・関東平野部(キャンプ場標高500m想定)の目安。標高が上がれば100mあたり0.6℃低下を加味する。

マットのR値: 地面からの冷気を切る

初心者がやりがちな最大の失敗: 寝袋だけを豪華にしてマットを軽視する。地面の冷たさは寝袋の下側を押し潰して保温材を無効化します。マット選びの基準がR値です。

R値の目安と対応季節
初心者は迷ったらR値3以上を選べば9月〜5月までカバーできる。

R値は足し算できる

マットは重ねればR値が加算されるのが便利な特性。例えば「夏用R1.5 + クローズドセルR2.0 = R3.5」で秋冬対応に格上げできます。1枚で全シーズンを狙うより、薄マット+厚マットを季節で使い分ける運用が経済的です。

マット3種の比較(エア/インフレータブル/クローズドセル)

キャンプ用マットは構造別に3タイプ。「快適性・軽量性・耐久性・価格」のトレードオフで選びます。

種類厚み重量R値の幅価格帯特徴
エアマット 5〜10cm 400〜600g R 1.5〜7 1〜3万円 寝心地最高、パンクリスク、ポンプ必要なことも
インフレータブル 3〜8cm 500〜900g R 2.0〜5 8千〜2万円 中にウレタンが入り自動膨張、バランス型
クローズドセル 1.5〜2cm 300〜450g R 2.0〜2.6 3千〜7千円 パンク無し、畳めない(嵩張る)、座布団兼用可
マット3種の立体イメージ
厚いほど寝心地は良いが、壊れるリスクも上がる。初心者はまずクローズドセル+インフレータブルの併用が鉄板。
Tips: 初心者の第一歩

迷ったらThermarest Z-lite Sol(R2.0、6千円、パンク無し)から始める。1枚で夏は足り、寒い日はエアマットに重ねる。雑に扱えるのも精神衛生上大事。

寝袋+マット+テントの「三位一体」

寝袋単体で「温度表記通りの性能」は出ません。マットで地面冷気を切り、テントで風を防ぎ、寝袋で体温を閉じ込める。この3つが揃って初めて快眠ラインに到達します。

Exercise 1
入門

「10月の奥多摩・標高800m」の装備を組む

10月中旬、奥多摩(標高約800m)のキャンプ場で1泊する想定で、寝袋のコンフォート温度とマットのR値の目安を決めましょう。

  1. 該当エリアの夜間最低気温を天気予報で調べる(想定: 平野部10℃ → 標高補正で約5℃)
  2. マージン5℃を加えて、必要な寝袋のコンフォート温度を決める
  3. マットのR値を決める
模範解答を見る

推奨: コンフォート0℃の寝袋 + R値3.0のマット

  • 現地最低気温 約5℃ に対し5℃のマージン → コンフォート0℃
  • 具体例: モンベル ダウンハガー650 #3(快適0℃)
  • マットは3シーズン万能のR3クラス(例: Z-lite Sol R2.0 + インフレータブルで重ね使い も可)
  • 失敗例: コンフォート10℃の夏寝袋を使うと、深夜2-3時に確実に目が覚める

ポイントは「現地気温+マージン5℃」と「標高100mあたり0.6℃」の補正。平野部の予報だけを鵜呑みにしないこと。

Exercise 2
応用

現在使っている(またはレンタル検討中の)寝袋を診断する

手持ちまたは気になっている寝袋1本を選び、以下を書き出してください。

  1. 形状(マミー/封筒/キルト)
  2. 保温材(ダウン/化繊、ダウンならFP、総重量)
  3. コンフォート温度/リミット温度
  4. その寝袋で使える「安全な最低外気温」(コンフォート+5℃)
  5. どの季節・エリアなら使えて、どこではマットやウェアで補う必要があるか
模範解答を見る

回答例: モンベル ダウンハガー650 #3

  • 形状: マミー(ストレッチ生地で寝返り可)
  • 保温材: 650FPダウン、総重量760g
  • コンフォート 0℃ / リミット -5℃
  • 安全な最低外気温: 5℃(コンフォート0℃ + マージン5℃)
  • カバー範囲: 本州の3-4月, 10-11月の平野〜低山キャンプ。真冬(1-2月)は湯たんぽ+インナーシーツ+R4以上のマットで補う必要あり。

この「安全な最低外気温」を頭に入れておくと、天気予報を見た瞬間に装備判断できるようになる。書き出しておくと家族とのキャンプ計画時にも便利。
