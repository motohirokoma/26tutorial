夜の過ごし方・快眠・防寒

ライト・レイヤリング・就寝時の寒さ対策

所要時間: 25分 Phase: Core 難易度: 初級
この章のゴール

日が落ちた後の「暗くて寒い」という2大不快要素を、ライト戦略とレイヤリングで攻略する。夕方〜就寝〜夜明けまでのルーティンを手順化し、春秋の寒い夜でも朝までぐっすり眠れる状態を作れるようになる。

夕方のルーティン：暗くなる前に片付ける

ソロキャンプの夜は日没から一気に加速する。17時に「まだ明るいな」と油断していると、気づけば18時には手元も見えないほど暗くなっている。明るいうちに「食べる・片付ける・暖める」を終わらせるのが快適な夜の基本ルールだ。

夕方〜就寝までのタイムライン(春秋モデル)
16:00 / 残り2時間
夕食の仕込み開始
米を研ぐ・食材カット。このタイミングで水も満タンに。
16:30 / 残り1.5時間
焚き火開始・調理
暗くなる前に火を安定させる。鍋・メスティンを火にかける。
17:30 / 残り30分
夕食・洗い物
食器類は明るいうちに全洗浄。残飯・ゴミは密閉袋へ。
18:00 / 日没
ランタン点灯・防寒着装着
冷え込みが始まる前に上着を1枚追加。ヘッドライトを首にかける。
21:00
焚き火納め・歯磨き
火を完全鎮火し、貴重品と食料をテント内またはクーラーへ。
22:00
就寝
着替え→寝袋へ。枕元にヘッドライトと水筒を配置。
「日没30分前には調理が終わっている」を逆算して16時から動く。
Tips

スマホの「日の入り」時刻を必ずチェックして、日の入りの2時間前にアラームをセットする。秋冬の山中は日没後の温度低下が劇的で、18時の快適と19時の「凍える」が隣接している。

ライト戦略：2系統で暗闇を攻略する

ソロキャンプの夜はライトの設計でQOLが大きく変わる。基本は「メインランタン(拠点照明)+ ヘッドライト(手元・移動)」の2系統。これを怠ると「暗闇でテントの出入りに手間取る」「手元が見えずお湯をこぼす」などの事故につながる。

  • メインランタン — サイト全体を照らす。テーブル上かタープの中心に吊るす。LED式が主流で、200-400ルーメンが目安
  • ヘッドライト — 両手を自由にしながら手元・足元を照らす。トイレ・洗い物・テント内で必須。150ルーメン前後がちょうどいい
  • サブランタン(任意) — テント内の間接照明。暖色系LEDやガス式の小型ランタンで雰囲気作り
  • モバイルバッテリー — 10000mAh以上。スマホ充電・ランタン充電の兼用で1つあると安心

ランタン配置(ソロサイト)

もっと詳しく: ルーメン(lm)とルクス(lx)の違い

ルーメン(lm)は光源が発する全光束の総量。ランタンのパッケージに書かれているのはこちら。対してルクス(lx)は1平方メートルあたりに届く光の量で、距離が離れると急激に減衰する。「200lmのランタンでも2m離れたテーブル上では50lx程度」というイメージ。吊るす高さ・距離で体感の明るさは大きく変わるので、ルーメン値だけでなく配置にも注意を。

Warning

就寝時にランタンを点けたままテント内に置いて眠るのは厳禁。ガス式はCO中毒の危険、LED式でも就寝中のバッテリー切れは翌日の行動に支障。必ず鎮火・消灯してから眠ること。

夜のやることリスト：静けさをどう味わうか

暗くて暇、ではない。ソロキャンプの夜こそ「何もしない贅沢」を味わうべき時間。ただ、何の準備もなく「何もしない」を迎えると寒さと退屈でテンションが下がる。以下のリストから2-3個を組み合わせると夜が楽しくなる。

防寒レイヤリング：3層で温度調節する

キャンプの防寒は1枚の分厚いコートではなく、レイヤリング(重ね着)で組む。ベース・ミドル・シェルの3層が基本構造だ。

3層レイヤリングの構造(身体の断面)
3層それぞれに役割がある。どれか1層欠けても寒さに耐えられない。
素材別の特性(体感評価)

メリノウールは価格こそ高いが防臭性が圧倒的で、連泊でも匂わない。ダウンは軽量で最強の保温だが濡れたら終わり。綿(コットン)は汗をかくと乾かずに冷えるので、下着含めキャンプでは避ける。

もっと詳しく: 「綿は死、化繊は友、ウールは神」の意味

登山やアウトドアでよく聞くスローガン「Cotton Kills」(綿は殺す)。綿は水を吸うと5倍以上の重量になり、かつ乾燥が遅い。汗で濡れた綿シャツを着続けると気化熱で体温が急速に奪われ、春秋でも低体温症のリスクがある。キャンプで「下着・靴下・Tシャツ・パーカー」すべてを化繊かメリノウールに置き換えるだけで、防寒性能が劇的に上がる。

就寝時の工夫：快眠を作る5つの技

寝袋の快適使用温度は、あくまで「若い男性が軽装で入った場合」の理想値。実際には表示温度+5度の想定で用意しておくと安心だ。

  • 就寝用の服に着替える — 日中着た服の汗が夜の冷えを招く。乾いたベースレイヤーとフリースに着替えて寝袋へ
  • 湯たんぽを仕込む — 焚き火で沸かしたお湯をナルゲンボトルに入れて寝袋内へ。足元・腰を直接温める
  • マットのR値を意識 — 地面からの冷えはマットでしか防げない。春秋はR値2以上、冬は4以上
  • 帽子・ネックウォーマー — 頭と首から体温の30%が逃げる。ニット帽+バフを就寝時も着用
  • 寝る前にトイレを済ます — 尿を温めるのにも体温を使う。空にしてから入ると寝袋の暖まりが早い
快眠を作る三位一体
どれか1つが欠けても朝までぐっすりは実現しない。

寒い夜の応急対応：持ち物で何とかする

想定外に冷え込んだ時、夜中の2時に目が覚めて震えが止まらない…という事態は誰にでも起こる。諦めずに「寝袋の中の断熱力」を増す応急処置で朝まで乗り切る。

寒さ応急対応チェックリスト(上から順に試す)
  • 貼るカイロを腰・足首に貼る — 皮膚直貼りは低温やけどの危険。肌着の上から。腰(仙骨)と足首(三陰交)が最も効く
  • 着ている服を全部寝袋内に入れる — 明日着る服も寝袋内へ。余計な空気空間を埋めると保温効率UP
  • 湯たんぽを作る — バーナーでお湯を沸かしナルゲンボトルへ(キャップ要確認、耐熱PC製で)
  • 寝袋のドローコードを絞る — フード周りの紐を顔周りだけ出るまで絞る。呼気で首周りを温めない
  • マットを2枚重ねる — 銀マット+エアマット等で地面からの冷えを遮断。シュラフカバーがあれば被せる
  • それでもダメなら起きて焚き火 — 我慢して低体温になる前に行動。車がある場合は車内に退避する勇気も
震えが止まらない/判断力が鈍るは低体温症の初期サイン。迷わず暖を取る。
Warning

貼るカイロは必ず衣服の上から貼る。就寝時は体が動かないため皮膚と密着して低温やけどのリスクが高い。また就寝中に顔の近くに置いて寝るのも避ける。

もっと詳しく: 低体温症の3段階サインと対応

軽度(35-33度): 震え・手指の動きが鈍い・早口で呂律が怪しい。→ 体を動かして熱を作る・温かい飲み物・乾いた服に着替える。
中度(33-30度): 震えが止まる・意識朦朧・判断力低下。→ 保温しつつ外部加温(湯たんぽ・体をこすらない・温かい甘い飲み物)。自力下山は避ける。
重度(30度以下): 意識消失・心停止の危険。→ 救助要請(119 / 山岳救助)。体を揺さぶらず慎重に搬送。
ソロキャンプで中度以上になったら迷わず119。「行ける所まで自分で」はやめる。

Exercise 1
入門

自分のライト装備を棚卸しして2系統化する

手持ちのライトを並べて、「メインランタン」「ヘッドライト」の2系統が揃っているかを確認してみましょう。

  1. 家にあるライト(スマホライト含む)をすべて机に並べる
  2. それぞれの明るさ(ルーメン)・電源方式・連続点灯時間を調べて表にする
  3. メイン(200lm以上)・ヘッド(150lm前後)の役割に分けて、不足しているものをリストアップ
  4. 不足分の予算感と買うべき順番を決める(例: まずヘッドライト→次にLEDメインランタン)
模範解答を見る

例: 棚卸し結果

  • 手元のiPhone → 50lm相当 / 常時 / 緊急用にのみ使う
  • 100均のLEDランタン → 80lm / 単3x3 / メイン不足(200lm以上の追加購入決定)
  • ヘッドライトなし → 最優先購入(Ledlenser MH5・約5,000円)

購入順: ①ヘッドライト(5,000円) → ②LEDメインランタン300lm(8,000円)。合計13,000円で夜のQOLが一気に上がる。

ポイント: ルーメン値はパッケージ・ECサイトに必ず書いてある。書いてない安価品は実測値を信用しない方がいい。

Exercise 2
応用

次回キャンプの「寒さ想定」を立てて装備を決める

次回のキャンプ予定地・日時の最低気温を調べて、レイヤリング・寝袋・就寝装備の組み合わせを計画してみましょう。

  1. 予定地・日付・標高を決める(例: 丹沢・4月20日・標高500m)
  2. tenki.jpや山の天気予報で予想最低気温を調べる(標高100mごとに気温-0.6度補正)
  3. 最低気温+5度を基準に、寝袋の快適温度を選定
  4. レイヤリング3層の具体的な服装を決める
  5. 想定外の寒さに備えた応急装備(カイロ・予備インナー)を1-2個追加
模範解答を見る

例: 丹沢 4月20日 標高500m

  • 予想最低気温: 平地8度 - 0.6度×5 = 約5度
  • 寝袋基準: 5度 + 5度 = 快適温度10度以下のものを選定(モンベル ダウンハガー650 #3等)
  • 就寝ウェア: メリノウール長袖+フリース+化繊タイツ+厚手ソックス+ニット帽
  • 夜の着衣: ベース(メリノ)+ミドル(フリース)+シェル(ウインドブレーカー)
  • 応急装備: 貼るカイロ×3、予備ソックス×1、ナルゲンボトル(湯たんぽ用)

ポイント: 標高補正(100mで-0.6度)、風の影響、放射冷却(晴天の朝は予報より2-3度低くなることも)を加味。春の山は「想定より1-2度寒い」想定が安全。
